# Project Setup Checklist

Follow the steps below before you open a pull request against the main branch.

- Install the recommended editor extensions listed in the contributing guide.
- Run the setup script to install dependences and configure git hooks.
- Copy the example environment file and fill in you local secrets.
- Run the test suite once locally to confirm a clean baseline.
- Read the style guide before making any large changes.

Once these steps are complete, request a review from a maintainer.
