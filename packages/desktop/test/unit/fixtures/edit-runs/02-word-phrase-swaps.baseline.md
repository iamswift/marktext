# Getting Started with Our Platform

In order to utilize the dashboard effectively, new users should first complete the setup wizard. Once you have done so, all of the core features become instantly available.

A large number of teams rely on the reporting tools due to the fact that manual tracking wastes time.

At this point in time, administrators can make a decision about which integrations to enable for the whole workspace.

If you need help, the support team is available around the clock and additionally provides live chat during business hours.

We recommend that you utilize the built-in templates prior to creating a custom workflow from scratch.
