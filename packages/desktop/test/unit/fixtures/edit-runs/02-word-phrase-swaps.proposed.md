# Getting Started with Our Platform

To use the dashboard effectively, new users should first complete the setup wizard. Once you have done so, all of the core features become instantly available.

Many teams rely on the reporting tools because manual tracking wastes time.

Right now, administrators can decide which integrations to enable for the whole workspace.

If you need help, the support team is available 24/7 and also provides live chat during business hours.

We recommend that you use the built-in templates before creating a custom workflow from scratch.
